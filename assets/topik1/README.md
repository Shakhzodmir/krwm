# TOPIK I (초급) — материалы

Сюда складываем **проверенные** исходники для уроков и тренажёров TOPIK I (уровни 1–2).
На основе этих данных строим материалы и уроки в приложении.

## Структура папок

| Папка | Что кладём |
|-------|-----------|
| `past-papers/` | Официальные прошлые тесты (기출문제): PDF вопросов + аудио + лист ответов |
| `listening/` | 듣기 — аудиофайлы (mp3) + скрипты |
| `reading/` | 읽기 — тексты, объявления, таблицы вопросов |
| `vocab/` | Списки лексики (어휘) — официальные и тематические |
| `grammar/` | Грамматика (문법) — список конструкций уровня 1–2 |
| `writing/` | 쓰기 기초 — правила письма, орфография, интервалы |

## Правила

- Только **проверенные** источники (официальные NIIED/기출, признанные учебники).
- Каждому файлу — понятное имя: `topik-NN-listening.mp3`, `topik-NN-reading.pdf`.
- Если источник из интернета — записать ссылку в `SOURCES.md`.

См. `SOURCES.md` — список найденных проверенных источников.
