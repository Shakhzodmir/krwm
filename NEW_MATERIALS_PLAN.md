# План адаптации новых материалов → приложение «Korean with Madie 🌸»

Источник: `assets/new material/` — 27 файлов (~1.1 ГБ). Документ — пошаговая дорожная карта: что с каждым материалом делать и в каком порядке.

---

## 0. Главные принципы (действуют на всех этапах)

1. **Формат всего контента — KO–RU.** Корейский → русский перевод, чередование (как в `toronpan ... docx`). Korean всегда с транскрипцией там, где это нужно новичку.
2. **Узбекский язык НЕ используем.** Из узбекских материалов берём только *темы и смысл*, итог оформляем на корейском + русском. Узбекский текст в приложение не попадает.
3. **Копирайт.** Официальные учебники (NIIED) и TOPIK — это *референс/основа* для нашего оригинального контента (структура тем, грамматика, типы упражнений — это методика). Страницы/тексты/картинки дословно в приложение не публикуем и в git не кладём.
4. **Размер/git.** Папка `assets/new material/` уже в `.gitignore`. PDF в репозиторий не идут.
5. **Стек.** Vanilla JS/HTML/CSS, без сборки. Каждая фича — адаптивна (375/768px) и в тёмной теме с первого дня.
6. **После каждого этапа:** `node --check app.js` → Playwright-smoke → бамп кэша `?v=` в `index.html` → список изменений + что проверить вручную.
7. **Инструмент извлечения:** для чтения PDF ставится `poppler` (`pdftotext`). Нужен на этапах 2, 3, 4 (извлечение текста из сканов/PDF).

## 2. Этап 1 — раздел «Чтение» + режим 말하기 (читать вслух)  ⏱ быстро, контент готов

**Источник:** `toronpan olkki perevod otveti.docx` — **5 готовых текстов** с KO-RU разбором (корейское предложение → перевод RU → разбор слов → грамматика), 24 предложения суммарно:
1. 신재생에너지 — возобновляемая энергия;
2. 한옥 — традиционный корейский дом (기와집·초가집·너와집, формы по регионам);
3. 위치 추적·119 구조 — отслеживание телефона и спасение + спор о приватности;
4–5. 합리적 소비 — рациональное потребление (траты в пределах дохода, приоритеты).

Это стартовый набор раздела; дальше в него добавляются новые тексты.

**Что делаем:**
1. Распарсить docx → массив `{ ko, ru, words:[{ko,ru}], grammar? }` по каждому предложению + общий перевод.
2. Завести структуру `READING_TEXTS = [ { id, title, level, ko, ru, sentences:[…], src } ]` в `app.js`.
3. **Раздел «Чтение» (독해):** список текстов → открыл и **просто читаешь**. Текст целиком на корейском, рядом/по тапу — перевод RU; по тапу на предложение раскрывается пословный разбор. Переключатель «показать/скрыть перевод».
4. **Режим 말하기 — читать вслух:**
   - **Образец:** кнопка ▶ озвучивает предложение носителем (Web Speech, ko-KR) — слушаешь, как звучит.
   - **Практика:** читаешь вслух за образцом, по предложениям; крупный текст, подсветка текущей строки, темп/скорость.
   - **Опционально (если поддерживается браузером):** распознавание речи (SpeechRecognition ko-KR) — сравнить прочитанное с текстом, подсветить совпадение. Если API нет — просто режим «читай вслух за диктором».
5. **Вход:** новый раздел в Уроки (или рядом с углублённым ТОПИК) — «📖 Чтение / 말하기». Адаптив + тёмная тема.
6. **Уточнить у пользователя:** этот перевод — его авторский (значит используем свободно)?

**Результат:** раздел «Чтение», где можно читать тексты KO-RU и тренировать чтение вслух (말하기). Текст про энергетику — первый, шаблон для всех будущих.

## 3. Этап 2 — TOPIK 읽기 + ключи → Mock-экзамены и углублённый раздел

**Источник:** `2교시_읽기_문제지.pdf` (вопросы) + `정답표_듣기_읽기.pdf` (ответы).

**Что делаем:**
1. Поставить `poppler`, `pdftotext` → извлечь текст 읽기 и таблицу ответов.
2. **Сверить**: не дубликат ли это одного из 9 тестов в `TOPIK2_EXAMS` (회차 41/47/52/60/64/83/91/96/102). Если новый — продолжаем.
3. Структурировать вопросы: `{ n, points, instr (RU), passage, options, answer, explain (RU/KO) }` — формат уже принят в `TOPIK2_EXAMS`.
4. Добавить как **новый mock-тест** в реестр экзаменов → автоматически:
   - появится в разделе **Экзамены (Mock)** с таймером 60 мин;
   - попадёт в **углублённый раздел** (вопросы разложатся по 18 типам 읽기 по номерам).
5. Написать русские разборы (`explain`) к каждому вопросу — почему верный/неверный, в нашем стиле ✅/✖️.
6. 듣기 — только если найдётся аудио + скрипт; иначе пометить «듣기 — скоро».

**Результат:** +1 реальный тест в Mock и в углублённый раздел.

## 4. Этап 3 — Учебники + 워크북 → новые уроки/модули и игры

**Источник:** серии `한국어 1-1…2-2`, `영어권 1-1…4-1` (본책 + 워크북 + 교사용 지도서), `6-1`.

**Подход (по уровням, от 1 к 6):** для каждого учебника извлекаем *методику*, не копируем страницы.

1. **Тематический план уроков** (단원/과): последовательность тем уровня → структура наших модулей.
2. **Лексика по урокам** → словарные наборы (KO-RU + транскрипция) для Уроков / Самостоятельного изучения.
3. **Грамматика урока** → грамматические карточки KO-RU (по образцу банков углублённого раздела).
4. **Упражнения из 워크북** → интерактивные задания и **мини-игры** (сопоставление, заполни пропуск, порядок слов, выбор). «Рисунки-задания» → визуальные игры.
5. **교사용 지도서** → методические подсказки → тексты-объяснения «как это работает» в уроках Madie.

**Структура каждого модуля урока (обязательно):**
- **① Урок** — объяснение темы + лексика + грамматика (KO-RU).
- **② Задание** — интерактивная отработка прямо в модуле (соответствует теме урока, из 워크북).
- **③ Домашка** — отдельный закрепляющий блок к этому же уроку (из 워크북), с отметкой «выполнено» и XP.

То есть каждому уроку соответствуют **своё задание и своя домашка** — связаны с темой модуля, не общие.

**Куда:** расширяем разделы **Уроки** (Первые слова, Простые предложения) и **Самостоятельное изучение**; задания-игры → раздел **Игры**.

> Весь итог — оригинальный контент в KO-RU по образцу учебника, а не его копия.

**Порядок внутри этапа:** 1-1 → 1-2 → 2-1 → 2-2 → (영어권 для расширения) → 6-1 в конце.
